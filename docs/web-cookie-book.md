# The Web Cookie Book

The Web Cookie Book is the learner path inside Safari Cookie Cleaner. It explains what cookies are, how they work, how they get stretched into tracking systems, and why browser storage often outlives a simple cookie delete.

This is not a claim that every cookie is malicious. Many cookies hold ordinary session or preference state. The important distinction is that the same mechanism can also carry identifiers that follow a person across visits, devices, and embedded services.

## What this path is for

Read this path if you want to:

- understand the difference between session state and tracking state
- inspect what a browser actually stores for a page
- see why deleting one cookie may not reset everything a site knows
- learn how first-party and third-party tracking overlap in practice

## What this path is not

This is not a comprehensive browser privacy textbook. It is a compact, practical guide tied to the kinds of storage and tracking behaviour Safari Cookie Cleaner is built to clean up.
